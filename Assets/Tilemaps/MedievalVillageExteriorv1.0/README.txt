=== USAGE TIPS ===
RPG Maker MZ & MV:
- Place tilesheet images A2, A3, and B in "img/tilesets" in your RPG Maker project's files - use the same or similar file name format.
- Place the wall sconce spritesheet PNG in "img/characters" in your RPG Maker project's files. Keep "!$" at the beginning of the file name for best results.
- In the Database menu, under "Tilesets", select just below the tilesheet type you want to import (for example, to import the ground tilesheet, click under "A2(Ground):" and select "MedievalVillage_A2").
- Create a new event for the sconce animation, and under the Event Editor window, select the square under "Image". Select the wall sconce spritesheet and hit "OK". Under "Autonomous Movement", set "Type" > "Fixed" and set "Speed" > "5: x2 Faster". Under "Options", select "Stepping" and "Direction Fix". Hit "Apply".

Other Engines/Raw Asset Use:
- Ensure that transparent pixels are maintained for best results.
- If resizing tiles, only scale UP, not down, for best quality.
